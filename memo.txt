* runlevel :
runlevel[n].target
0 : poweroff.target : shot down and power off
1 : rescue.target : rescue shell
2~4 : multi-user.target : start non-graphical multi-user shell
5 : graphical.target : set up a graphical multi-user shell
6 : reboot.target : reboot
